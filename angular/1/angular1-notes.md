**What is AngularJS?**

* JavaScript framework
* For creating Single Page Applications (SPAs)

**Why Use AngularJS?**

* Advantages
  - Excellent templating engine
  - Handles DOM "masterfully"
  - Easy data manipulation
* MVC Architecture
  - Models
  - Views
  - Controllers
* Core Features
  - Directives
    + Basically, commands that start with "ng-"
  - Data binding
  - Filters
  - Modules
  - Routes
  - Controllers
