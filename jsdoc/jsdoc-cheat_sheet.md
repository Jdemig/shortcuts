## Types

### Common Types

JavaScript has many types that can be used with various JSDoc tags that accept type information.  The following is a
list of some of the most common types (using the expected capitalization):

- Arguments
- Array
- boolean
- Date
- Function
- Mixed
- null
- number
- Object
- RegExp
- string
- undefined
