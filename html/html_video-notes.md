# HTML Video Notes


## Browser Support

There are 3 supported video formats: MP4, WebM, and Ogg.

| Browser           | MP4 | WebM | Ogg |
| ----------------- | --- | ---- | --- |
| Internet Explorer | YES | NO   | NO  |
| Chrome            | YES | YES  | YES |
| Firefox           | YES | YES  | YES |
| Safari            | YES | NO   | NO  |
| Opera             | YES (from Opera 25) | YES  | YES |
