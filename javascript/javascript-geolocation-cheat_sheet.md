# JavaScript Geolocation Cheat Sheet

See https://developer.mozilla.org/en-US/docs/Web/API/Geolocation/Using_geolocation
and https://developer.mozilla.org/en-US/docs/Web/API/Geolocation for details.

The geolocation API allows the user to provide their location to web
applications if they so desire.  For privacy reasons, the user is asked for
permission to report location information.
